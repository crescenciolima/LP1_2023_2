# LP1_2023_2
Código desenvolvido na disciplina Linguagem de Programação 1 (LP1) semestre 2023.2

Neste repositório você vai encontrar os códigos produzidos durante os encontros síncronos da disciplina LP1 do Instituto Federal de Educação, Ciência e Tecnologia da Bahia (IFBA) campus Vitória da Conquista semestre 2023.2 do curso Bacharelado em Sistemas de Informação (noturno). 

Os vídeos com as gravações dos encontros síncronos realizados durante o período de modalidade de Atividades de Ensino Não Presenciais Emergenciais (AENPE) podem ser encontrados no link a seguir:
